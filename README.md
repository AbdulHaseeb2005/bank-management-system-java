# Bank Management System (Java Swing)

## Overview
This project is a GUI-based Bank Management System developed using Java Swing. It simulates essential banking operations through multiple interactive screens and demonstrates core Object-Oriented Programming (OOP) concepts.

## Features
- User Authentication (Login & PIN)
- Multi-step Account Registration
- Deposit Money
- Withdraw Money
- Fast Cash Option
- Balance Enquiry
- Mini Statement
- Navigation across multiple GUI screens

## Application Screens / Modules
The system consists of the following components:

- Login  
- Signup  
- Signup2  
- Signup3  
- main_Class  
- Deposit  
- Withdrawal  
- FastCash  
- BalanceEnquriy  
- Pin  
- mini

- pr_page  

## Technologies Used

- Object-Oriented Programming (OOP)
